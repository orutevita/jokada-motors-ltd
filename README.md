# JOKADA MOTORS LIMITED

A professional motor vehicle dealership website designed to showcase available vehicles, facilitate customer inquiries, and provide a seamless user experience across desktop and mobile devices.

- Responsive and modern user interface
- Vehicle inventory management
- Detailed vehicle listings
- Customer inquiry forms
- Secure administration portal
- Contact and location information
- Payment integration support
- Search and filtering functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Python

## Project Structure

- Backend (Python)
- Vehicle database management
- Administrative dashboard
